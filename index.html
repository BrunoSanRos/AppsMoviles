Validador de DNI (España)

Introduce el número de DNI y la letra para comprobar si coinciden. (Validación cliente con JavaScript)

Debe ser un número entre 0 y 99,999,999 (sin espacios ni puntos).

Cálculo: resto = dni % 23 → letra = letras[resto]. Array de letras utilizado por el script.
